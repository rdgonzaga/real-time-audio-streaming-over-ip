# Real-Time Audio Streaming over IP

CHIU, Lance & GONZAGA, Rainer

## Project Files

- `peer.py` — main application logic and command loop
- `sip.py` — SIP message building/parsing and SDP handling
- `media.py` — RTP and RTCP packet handling and media loops
- `audio.py` — WAV reading, microphone capture, and playback
- `utils.py` — helper functions
- `sample.wav` — sample audio file for testing

## Requirements
 - Python 3.10 or later
 - Required Python packages listed in `requirements.txt`
